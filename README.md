# Hermes Agents

Dark, techy lead-generation landing page for deploying Hermes-based autonomous AI teams.

## Stack

- React + TypeScript
- Vite
- Static Vercel deployment

## Development

```bash
npm install
npm run dev
npm run build
```

## Positioning

Hermes Agents helps small teams deploy specialized AI agent profiles — Sales, Tech, Data, Ops, Marketing, Product — connected to Slack, Telegram, GitHub, Vercel and local/cloud infrastructure.
